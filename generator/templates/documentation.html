Everything you need to know to generate the perfect README for your project.

How It Works

Our generator follows a simple three-step process:

  1. Ingest: We analyze the files in your public GitHub repository to understand its structure, languages, and dependencies.
  2. Analyze: Our AI engine processes this information, along with any custom instructions you provide, to create a detailed project profile.
  3. Generate: Using this profile, the AI writes a clean, comprehensive, and well-formatted `README.md` file tailored to your project.

Using the Generator

1. Enter URL: On the home page, paste the full URL of your public GitHub repository.
2. Select Model: Choose between `Gemini` (balanced) or `Groq` (fast) AI models.
3. Add Instructions (Optional): Use the custom instructions box to guide the AI. For example: "Focus on the data processing pipeline" or "Add a section on API endpoints."
4. Generate: Click the "Generate README" button and wait for the magic to happen.

Editing & Exporting

After generation, you'll be taken to the result page. From there, you can:

  • Preview the rendered HTML.
  • View the raw Markdown source.
  • Copy the markdown to your clipboard.
  • Download the `README.md` file directly.
  • Edit the content in our live editor to make final adjustments.
