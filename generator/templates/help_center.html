Help Center

Have questions? We've got answers. Here are some frequently asked questions.

Is this service free to use?

Yes, our basic service is completely free for open-source and personal projects, with a generous daily limit. We plan to offer paid plans with higher limits and advanced features in the future.

What AI models are used?

We use Google's `Gemini-1.5-Flash` for its strong analytical capabilities and `Llama 3` via the Groq API for incredible speed. You can choose which model to use on the home page.

Does it work with private repositories?

Currently, the tool only supports public GitHub repositories due to API limitations. Support for private repos via GitHub App authentication is on our roadmap.

The generated README has some inaccuracies. What can I do?

AI is not perfect. If you find any issues, you can use the 'Edit' feature on the results page to make corrections. You can also try regenerating with a more specific custom prompt to guide the AI better.

How is my data used?

We fetch your code for analysis and pass it to the AI provider (Google or Groq). We cache the generated READMEs to speed up subsequent requests, but we do not store your code or use it for any other purpose.
